# Code of Conduct

WIGGWIGG is built by people who care about privacy and about treating each other well.
We want this project to be a place where anyone can contribute or raise a concern without
friction.

This project adopts the
[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its Code of Conduct. The short version: be respectful, assume good faith, and help us
keep the space welcoming. The full text at the link above governs.

## Reporting

To report a concern, contact **conduct@wiggwigg.ca**. Reports are handled confidentially,
and we respond as promptly as we can.
